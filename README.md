# 🎨 Design Patterns in C#

This is a learning repository that demonstrates various design patterns in real-world .NET implementations.

## 🧠 Covered Patterns
### Creational
- Factory
- Abstract Factory
- Builder

### Structural
- Adapter
- Decorator
- Proxy

### Behavioral
- Strategy
- Command
- Observer

## 🧪 Testing
Each pattern has unit tests using xUnit.
